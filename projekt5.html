Li-Ion Degradation

Prognose von Kapazität und Innenwiderstand basierend auf dem NASA Battery Dataset.

Data NASA Battery Dataset
Stack Quarto, Scikit-learn, Plotly

Analysis Overview

Dieses Tool visualisiert die Daten, die wir zu Beginn dieser Arbeit ausgewertet haben. Zusätzlich haben wir ein Modell trainiert, das mithilfe linearer Regression Vorhersagen zur Kapazität und zum Innenwiderstand ermöglicht. So erhält man einen Einblick, wann im Lebenszyklus eine Akkuzelle der größten thermischen Belastung ausgesetzt ist.
