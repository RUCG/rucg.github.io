Analyse und Prognose von Lithium-Ionen-Zellen

Dieses Tool visualisiert die Daten, die wir zu Beginn dieser Arbeit ausgewertet haben. Zusätzlich haben wir ein Modell trainiert, das mithilfe linearer Regression Vorhersagen zur Kapazität und zum Innenwiderstand ermöglicht. So erhält man einen Einblick, wann im Lebenszyklus eine Akkuzelle der größten thermischen Belastung ausgesetzt ist und wie sich die Zelle verhält, nachdem sie eine Kapazität von 70% RUL unterschreitet.

Technische Details

Technologien:
Quarto, Python
Bibliotheken:
Pandas, NumPy, Scikit-learn, Plotly, Matplotlib
Datenquelle:
NASA Battery Dataset
Fokus:
Datenvisualisierung, thermische Belastung, Batteriealterung, Prognose
